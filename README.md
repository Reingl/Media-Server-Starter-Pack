# Media-Server-Starter-Pack

This is a starter pack and my own personal archive on how i started my first server that is available to the public using Navidrome and Tailscale

### Here are a few requirements that are defo needed 

## Requirements

 1. Navidrome
 2. Tailscale
 3. Your own Navidrome config 

# Installation

## Step 1. Install for your own distro

## Arch Linux
### tailscale ver
```sudo pacman -S navidrome tailscale```
### cloudflare ver
```sudo pacman -S cloudflare navidrome```

## Debian/Ubuntu linux
```sudo apt install navidrome tailscale```
### cloudflare ver
```sudo apt install navidrome cloudflare```

## Step 2. clone this repo

```git clone https://github.com/Reingl/Media-Server-Starter-Pack.git```
